# Locally Usage
## Prerequisites
### OS
- Ubuntu 20.04 LTS+
- Windows 10

### Software
- Python 3.10+
- Python pip
- Python venv
- Python Poetry

## Install locally

Run this command at root of project
```shell
poetry install
```

# Containerized mode
you can omit all pre-requisites and run project with docker containers
- Docker engine 24.0.7+
